# Core Gulp Tasks

**Note:** Node 0.12 or higher is required.

## Usage

```
npm install --save-dev cloudfour/core-gulp-tasks.git
```

```js
var gulp = require('gulp');
var tasks = require('core-gulp-tasks');

tasks.css(gulp, {
  // Task options
});

tasks.js(gulp, {
  // Task options
});
```

```
gulp --tasks
```
